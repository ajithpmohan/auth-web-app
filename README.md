# 🔐 Next.js User Auth Frontend

A simple authentication frontend built with Next.js.
Includes a full authentication experience with registration, login, password recovery, and user profile management.

## 🚀 Tech Stack

- **Next.js**
- **TypeScript**
- **Tailwind CSS**

## 🔗 Backend Repository

This frontend integrates with the authentication API:

👉 https://github.com/ajithpmohan/user-auth-api

## ⚙️ Clone the repository

```bash
git clone https://github.com/ajithpmohan/auth-web-app.git
```

## 🌍 Environment Variables

This project uses environment variables to configure the backend API URL.

### Create a `.env.local` file in the root directory

```env
NEXT_PUBLIC_API_URL=http://localhost:5000
```
